Vue入门

MVVM模式

  • M:即Model,模型,包括数据和一些基本操作
  • V:即View,视图,页面渲染结果
  • VM:即View-Model,模型与视图间的双向操作(无需开发人员干涉)

在MVVM之前,开发人员从后端获取需要的数据模型,然后要通过DOM操作Model渲染到View中.而后当用户操作视图,我们还需要通过DOM获取View中的数据,然后同步到Model中.

而MVVM中的VM要做的事情就是包DOOM操作完全封装起来,开发人员不用再关注Model和View之间是如何互相影响的:

  • 只要我们的Model发生了改变,View上自然就会表现出来.
  • 当用户修改了View,Model中的数据也会跟着改变.

把开发人员从繁琐的DOM操作中解放出来,把关注点放在如何操作Model上.

什么是指令?

指令(Directives)是带有v-前缀的特殊属性.例如前面的v-model,代表双向绑定.

格式:

说明:

  • 该表达式支持JS语法,可以调用JS内置函数(必须有返回值)
  • 表达式必须有返回结果.例如1+1,没有结果的表达式不允许使用,如:var a=1+1;

插值闪烁

使用双花括号方式在网速较慢时会出现问题.在数据未加载完成时,页面会显示双花括号,加载完毕后才显示正确数据,我们成为插值闪烁.

v-text和v-html

使用v-text和v-html指令来代替双花括号

说明:

  • v-text:将数据输出到元素内部,如果输出的数据有HTML代码,会作为普通文本输出
  • v-html:将数据传输到元素内部,如果输出的数据有HTML代码,会被渲染.

v-model

v-text和v-html可以视为单项绑定,数据影响了视图渲染,反过来则不行,而v-model是双向绑定,视图和模型之间会互相影响.

既然是双向绑定,一定是可以在视图中可以修改数据,这样就限制了元素类型.目前v-model的可使用元素有:

  • input
  • select
  • textarea
  • checkbox
  • radio
  • components(Vue中的自定义组件)

基本上除了最后一项,其他的都是表单输入项.

多个checkBox对应一个model时,model的类型是一个数组,单个checkBox值是boolean类型

radio对应的值是input的value值

input和textarea默认对应的model是字符串

select单选对应字符串,多选对应也是数组

v-on(事件绑定)

基本用法

v-on指令用于给页面元素绑定事件

语法:v-on:事件名="js片段或者函数名

简写语法:@事件名="js片段或函数名"

事件修饰符

修饰符由点开头的指令后缀来表示

  • .stop:阻止事件冒泡(触发外层事件).

  • .prevent:阻止默认事情发生

  • .capture:使用事件捕获模式

  • .self:只有元素自身触发事件才执行.(冒泡或捕获都不执行)

  • .once:只执行一次

v-for(循环遍历)

基本用法

语法:v-for="数组元素别名 in 数组"

数组角标

语法:v-for="(数组元素别名,角标) in 数组"

角标为迭代索引,从0开始

同时也可以遍历对象,取出的为对象的每个属性,这里加角标会返回属性的名称

遍历数字

语法:v-for="别名 in 数字

遍历出来是1到N(数字值)

v-if

基本使用

v-if,条件判断,当结果为true时,所在元素才会被渲染.

语法:v-if="布尔表达式"

与v-for结合

当v-if和v-for出现在一起时,v-for优先级更高,也就是说,会先遍历,再判断条件.

v-else

在使用中可以使用v-else,v-else-if和v-if组成if-else组合.

注意:v-if和v-else的两对标签之间不能由其他标签

v-show

和v-if在功能上相似(控制元素是否在页面上显示),区别在于:v-show是控制display让元素隐藏,而v-if是直接将元素从页面上删除了.

相比之下.v-show性能更好,而v-if更安全.

v-bind

由于插值表达式在标签的属性中不能使用,所以为了可以修改属性,需要将要修改的属性变成vue的特殊属性.

语法:v-bind:属性名="Vue中的变量"

或::属性名="Vue中的变量"

定义全局组件

  • 组件其实也是一个Vue实例,因此他在定义时也会接收:data,methods,生命周期函数等
  • 不同的是组件不会与页面的元素绑定,否则就无法复用了,因此没有el属性.
  • 但是组件渲染需要html模板,所以增加了template属性,值就是HTML模板
  • 全局组件定义完毕,然和vue实例都可以直接在HTML中通过组件名称来使用组件了
  • data的定义方法比较特殊必须是一个函数.
