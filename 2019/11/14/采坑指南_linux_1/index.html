ssh免密登录复制密钥时没反应

问题描述

ssh免密登录复制密钥时没反应

错误信息:

今天在配置免密登录时,执行ssh-copy-id 主机地址时没有反应,检查了半天,原因如下

主要原因

静态地址和默认网关不匹配

解决办法

在控制台(Xshell)中输入setup,打开工具选项=>网路配置=>设备配置,选择你使用的设备.

将默认网关IP的前三位置为和静态IP一样的数值.

最后一位置为2.

保存并退出.

重启网络service network restart

注意:一定要重启!!!!!,否则还是连不上
